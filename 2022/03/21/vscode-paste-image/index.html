vscode配置paste image

2022-03-21

前言

最近感觉用vscode编辑Markdown也是十分清爽的，而写笔记少不了图片，我就想如何实现方便快捷地管理图片

需求

个人是在用snipaste截图

想达到的效果为

  1. 截图到剪贴板
  2. 直接在vscode中使用快捷键粘贴进来
  3. 可以自定义图片粘贴后保存的路径，方便管理
  4. 可以自定义图片的名称

插件

经过搜集资料，发现vscode中的插件paste image，这个插件使用快捷键可以把剪贴板里的图片直接粘贴到md中并保存本地文件

Type: “Paste Image” or you can use default keyboard binding: Ctrl+Alt+V (Cmd+Alt+V on Mac).

默认快捷键：
win：ctrl + alt + v
mac：command + alt + v

自定义配置

可以在paste image的介绍页看到github的地址

readme文件中已经详细介绍了基础配置

在vscode设置中找到paste image配置

具体每项配置就不再赘述了，仅记录自己用的一些

修改图片默认名称

修改文件保存路径，我没有修改这里的配置，使用了文件同名目录

${currentFileNameWithoutExt}-images

这里我把图片保存到md文件同级的一个同名文件夹后面加 - images的文件目录里

md文件插入图片的形式为：

[](图片地址)

使用快捷键粘贴图片后，插件其实是帮我们做了两件事

  1. 把图片保存到我们配置的目录中，以我们配置的名字保存
  2. 在md文件中，插入上述的图片形式，图片地址部分为图片路径
    • 我们可以在插入的地址文本前加上 ./ ，同样是配置，如果需要的话

这样就实现了使用paste image自定义图片保存路径和名称
